# 🔮 神秘塔罗占卜

一个精美的塔罗牌占卜网页，包含 22 张大阿卡纳牌的完整解读。

## ✨ 功能特点

- **22 张大阿卡纳牌**：完整的塔罗牌数据，包含每张牌的含义和解读
- **三张牌阵**：随机抽取三张牌，分别代表过去、现在、未来
- **精美界面**：神秘的紫色渐变设计，魔法粒子效果
- **卡牌翻转动画**：3D 翻转效果，增强交互体验
- **完整解读**：每张牌都有详细的含义说明
- **响应式设计**：适配桌面和移动设备

## 🚀 使用方法

直接在浏览器中打开 `index.html` 文件即可使用。

1. 点击"抽取牌阵"按钮
2. 系统会随机抽取三张牌
3. 牌面会自动翻转显示
4. 查看下方的完整解读
